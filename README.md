# React Shop

This is a simple e-commerce app built with React and webpack fully configured.

## How to use

    1. Clone the repository
    2. Install dependencies
    3. Run the app

### Steps

First
```bash
git clone https://github.com/NathanLazo/react-shop.git
```
CD into folder
```bash
cd react-shop
```
Then
```bash
npm install
```
Finally
```bash
npm run start
```
